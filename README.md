## 📋 What is codemoot?

codemoot is a tool designed to help teams build software together by using AI models. It lets you and your teammates chat, debate, and review code with the help of smart assistants named Claude and GPT. You do not need to know how to program to use codemoot. It structures collaboration in an easy-to-understand way and helps improve code quality by sharing ideas and checking work automatically.

---

## 💻 System Requirements

To use codemoot on your computer, make sure you have:

- A Windows 10 or later, macOS 10.14 or later, or a Linux operating system.
- At least 4 GB of free storage space.
- Minimum 8 GB of RAM for smoother operation.
- An internet connection to access AI features.
- A modern web browser or the codemoot application installed.

---

## 🛠 Features Overview

- **AI-Assisted Code Discussion:** Claude and GPT debate code changes and different solutions side by side.  
- **Collaborative Editor:** Work with your team on code projects in real time, no technical setup needed.  
- **Review and Feedback Automation:** AI reviews your code and points out possible bugs or improvements before you finalize.  
- **Project Tracking:** Keep all comments, suggestions, and code changes in one place for easy tracking.
- **Easy Interface:** The app uses simple menus, buttons, and chat windows without requiring knowledge of coding commands.

## 🔍 How to Use codemoot

Once you open codemoot, you will see a simple dashboard that guides you through starting or joining a project.

1. **Create a new project:** Click “New Project” to start from scratch.
2. **Invite collaborators:** You can add friends or team members by typing their email or sharing a link.
3. **Add your code or ideas:** Use the editor to type or paste your work.
4. **Start a debate:** Ask Claude and GPT to discuss differences in your code or how to improve it.
5. **Review suggestions:** Watch the AI point out errors or better ways to write your code.
6. **Make changes:** Edit according to feedback or come up with new ideas.
7. **Save and share:** Keep your progress saved on the cloud and share updates with your team.
